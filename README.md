# CV_CT

### 为了清晰我负责的模块，这里做一些GIF图片用于说明
## 广告模块
#### 广告是一个非常低耦合的模块，计划封装成SDK，携程旗下其他APP，智行，途家等会相继接入。对稳定性，兼容性要求很高，并有一个独立的后台支持。
### Banner广告
#### 功能方面：包含图片文字叠加自定义，iPhoneX尺寸适配，PageControl自定义，埋点数据监控，已完成酒店，旅游，机票等全公司十几个部门接入
